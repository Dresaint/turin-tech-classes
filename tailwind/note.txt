STEPS IN USING TAILWINDCSS
- make sure you have node js install on your vs code
- node -v (to confirm)
- npm int -y ( to )
- visit tailwindcss.com
-click on Docs
- click on Tailwind CLI
- Follow these steps:
    * npm install tailwindcss @tailwindcss/cli
    * create a css folder and create input.css file inside of the folder 
    * paste this code inside the input.css = @import "tailwindcss";
    * run - npx @tailwindcss/cli -i ./css/input.css -o ./css/output.css --watch
    * link your css folder/file to all the html files
        example:
         <link rel="stylesheet" href="./css/output.css">

    * go into your package.json and edit the script
        example:
        "scripts": {
            "build-css": "npx @tailwindcss/cli -i ./css/input.css -o ./css/output.css --watch"
        },

    * you can always run the tailwind with npm run build-css on yout terminal whenever the server is been stopped
