        CLONNING
    To clone a specific branch on GitHub is:
    git clone "url"

        TO CLONE A SPECIFIC BRANCH ON GITHUB:
    git clone -b branch-name --single-branch repo-url

        IF YOU WANT TO PULL FROM ANOTHER BRANCH APART FROM MAIN :
        git pull origin "branch-name"
